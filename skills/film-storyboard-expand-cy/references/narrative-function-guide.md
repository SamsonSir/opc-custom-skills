# 叙事功能分类与分镜展开策略

## 6种叙事功能

### 1. 建立场景（Establishing）

**定义：** 交代故事发生的时间、地点、世界观，让观众进入叙事空间。

**视觉识别特征：**
- 景别偏大（远景/全景为主）
- 人物占比小或不在画面中
- 环境细节丰富，有明显的时代/地域特征
- 光线通常交代时间段（晨光/午后/黄昏/夜晚）

**3分镜展开策略：**
- **空间交代（远景/大远景）：** 拉到更远的视角，展示场景所在的更大地理环境——城市天际线、山谷全貌、建筑群外观。用航拍或极高机位，强调世界观的规模感。
- **情绪特写（近景/特写）：** 聚焦场景中最能定义这个世界的一个细节——墙上的裂缝、窗台的雨滴、桌面的旧物件。通过微观视角传递空间的情感温度。
- **叙事推进（自由景别）：** 展示即将进入这个空间的人物，或空间中暗示即将发生事件的线索——远处走来的身影、桌上未读的信、打开的门。

---

### 2. 人物出场（Introduction）

**定义：** 核心角色首次亮相，建立观众对人物的第一印象。

**视觉识别特征：**
- 人物是画面绝对主体
- 景别通常在中景到全景之间
- 人物姿态/表情有性格暗示
- 可能有标志性道具或服装特征

**3分镜展开策略：**
- **空间交代（远景/大远景）：** 展示人物所处的完整环境，通过环境反衬人物身份——豪华办公室暗示权力、破旧小巷暗示困境、实验室暗示学者。人物可以是画面中一个小身影。
- **情绪特写（近景/特写）：** 捕捉人物最具识别性的细节——眼神中的某种光芒、嘴角的弧度、手上的伤疤、标志性饰品。用浅景深突出细节，传达内心状态。
- **叙事推进（自由景别）：** 展示人物即将做的事或正在进行的动作——起身走向某处、拿起某件物品、转身面对某人。暗示故事的方向。

---

### 3. 情绪高潮（Climax）

**定义：** 情感张力最强的时刻，角色内心的爆发或压抑达到顶点。

**视觉识别特征：**
- 表情/肢体语言强烈
- 光影对比可能极端（高反差或极度柔和）
- 构图可能打破常规（荷兰角、极端景别）
- 色调可能偏向饱和或极度去饱和

**3分镜展开策略：**
- **空间交代（远景/大远景）：** 用极端远景展示人物在环境中的渺小，或用环境的压迫感映射内心——暴风雨中的海岸、空旷的大厅、被火光映红的天空。环境成为情绪的外化。
- **情绪特写（近景/特写）：** 极端近距离捕捉情绪的物理表现——颤抖的嘴唇、含泪的眼睛、握紧的拳头、深呼吸时胸腔的起伏。用缓慢推进或固定长镜头，让情绪充分释放。
- **叙事推进（自由景别）：** 展示高潮之后的即刻反应——转身离去、做出决定性动作、瘫坐下来、破涕为笑。这是情绪的出口和故事的转向。

---

### 4. 转折时刻（Turning Point）

**定义：** 叙事方向发生变化的关键节点，信息揭露或事件突发。

**视觉识别特征：**
- 画面中存在"变化"的元素（新人物、新信息、意外事件）
- 人物表情从一种状态转向另一种
- 可能有强烈的光线变化
- 构图可能暗示对立或分裂

**3分镜展开策略：**
- **空间交代（远景/大远景）：** 展示转折发生前的"平静"全貌，或展示转折来源的方向——远处逼近的车灯、地平线上的烟雾、窗外突然变暗的天空。用反差制造悬念。
- **情绪特写（近景/特写）：** 捕捉人物意识到变化的瞬间——瞳孔微缩、手指停顿、呼吸暂停。用极短的停顿和精确的面部细节传达"一切都变了"的感觉。
- **叙事推进（自由景别）：** 展示转折的直接后果——人物开始新的行动、拿起或放下某件关键道具、画面切向转折的源头。节奏从停顿急转为运动。

---

### 5. 日常段落（Daily Moment）

**定义：** 角色日常生活的片段，节奏舒缓，用于展示性格和建立情感连接。

**视觉识别特征：**
- 自然放松的姿态和表情
- 日常环境（家、咖啡厅、街道）
- 光线自然柔和
- 色调温暖或中性
- 无明显冲突或张力

**3分镜展开策略：**
- **空间交代（远景/大远景）：** 展示人物生活的完整空间——街道全景、房间全貌、窗外的城市。用自然光和舒缓运镜，传达生活的质感和日常的韵律。
- **情绪特写（近景/特写）：** 聚焦日常中最有温度的细节——咖啡杯上的水雾、翻书的手指、阳光照在脸上的光斑。这些微小细节构建亲密感和在场感。
- **叙事推进（自由景别）：** 展示日常动作的延续或微小变化——走出家门、收到一条消息、窗外天色渐暗。用时间流逝暗示生活在继续。

---

### 6. 冲突对峙（Confrontation）

**定义：** 两个或多个力量之间的直接对抗，可以是人与人、人与环境、或人与内心。

**视觉识别特征：**
- 画面中有明显的对立元素（两人面对面、人物面对困境）
- 构图强调张力（对角线、分割构图）
- 光影对比强烈
- 人物肢体语言紧绷
- 可能有动态模糊或运动感

**3分镜展开策略：**
- **空间交代（远景/大远景）：** 展示对峙双方在空间中的位置关系——两端对立的身影、被战场分隔的两方、狭窄空间中的逼仄感。用构图本身表达对抗结构。
- **情绪特写（近景/特写）：** 交替捕捉对峙双方的面部——一方的坚定与另一方的犹豫、攻击者的眼神与防御者的表情。如果是人与自我的冲突，聚焦内心矛盾的物理表现。
- **叙事推进（自由景别）：** 展示冲突的升级或打破——第一个动作的发起、沉默被打破的瞬间、一方退让或倒下。用运动镜头和景别快速切换增强动感。

---

## 判断决策树

按以下顺序判断原图的叙事功能：

```
1. 画面中是否有明显的对立/紧张元素？
   ├─ 是 → 冲突对峙
   └─ 否 → 继续

2. 人物表情/肢体是否有强烈情绪？
   ├─ 是 → 情绪高潮
   └─ 否 → 继续

3. 画面中是否有"变化"或"意外"的视觉暗示？
   ├─ 是 → 转折时刻
   └─ 否 → 继续

4. 人物是否是首次出现/具有强烈的"亮相"感？
   ├─ 是 → 人物出场
   └─ 否 → 继续

5. 人物状态是否自然放松、处于日常活动中？
   ├─ 是 → 日常段落
   └─ 否 → 继续

6. 画面是否以环境为主、人物占比很小或不在画面中？
   ├─ 是 → 建立场景
   └─ 否 → 根据综合判断选择最接近的类型
```

### 快速判断辅助表

| 关键线索 | 指向的叙事功能 |
|---------|--------------|
| 景别大、人物小、环境丰富 | 建立场景 |
| 人物居中、姿态有个性、首次感 | 人物出场 |
| 表情激烈、光影极端、构图破格 | 情绪高潮 |
| 新元素出现、表情转变、光线突变 | 转折时刻 |
| 姿态放松、自然光、日常环境 | 日常段落 |
| 对立构图、肢体紧绷、张力空间 | 冲突对峙 |
