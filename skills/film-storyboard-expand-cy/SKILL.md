# 电影分镜拓展提示词

根据用户提供的参考图片，以电影分镜师视角深度分析后，扩展出3个不同叙事功能的分镜提示词。

**核心理念：这是图生图（img2img）工作流。** 每段提示词都必须在开头明确指示"以参考图中的人物/场景为基础"，让 AI 生成工具知道要基于原图来变换镜头，而不是从零生成。提示词本身就要包含对原图的引用指令。

---

## 工作流

### 第 1 步：图片深度分析（锚定镜头提取）

仔细观察用户提供的图片，按以下维度提取视觉信息，形成"锚定镜头"参数表。后续所有分镜必须与这些参数保持一致。

| 维度 | 提取内容 |
|------|---------|
| 画面比例 | 原图宽高比（如 16:9、4:3、1:1、9:16、3:4 等），后续生成必须保持一致 |
| 人物外貌 | 性别、年龄感、脸型、体型、发型/发色 |
| 服装 | 材质、颜色、款式、配饰、磨损程度 |
| 场景/世界观 | 时代、地点类型、环境细节（建筑、植被、天气等） |
| 视觉风格 | 写实/动画/赛博朋克/油画等、渲染质感（胶片/数码/3D等） |
| 色调 | 冷暖倾向、饱和度、主色调、色彩分级名称（如 teal and orange） |
| 光线 | 光源方向、色温、明暗对比度、光线类型（伦勃朗光/蝴蝶光/自然光等） |
| 构图 | 当前景别、人物位置、前景/背景关系 |
| 道具 | 关键物件的材质、颜色、状态 |
| 情绪氛围 | 画面传达的情感基调（孤独/温暖/紧张/宁静等） |

将分析结果以简洁表格形式输出给用户，作为锚定镜头记录。

### 第 2 步：判断叙事功能

读取 `references/narrative-function-guide.md`，根据判断决策树分析原图在叙事中的角色：

- 建立场景（Establishing）
- 人物出场（Introduction）
- 情绪高潮（Climax）
- 转折时刻（Turning Point）
- 日常段落（Daily Moment）
- 冲突对峙（Confrontation）

基于判断结果，确定3个分镜的具体展开方向。将判断过程和结论简要告知用户。

### 第 3 步：生成3个分镜提示词

读取 `references/shot-language-reference.md`，按以下固定角色分工生成3段提示词：

**分镜角色分工：**
- **分镜1 — 空间交代：** 远景/大远景，揭示完整空间与世界观
- **分镜2 — 情绪特写：** 近景/特写，捕捉人物内心状态
- **分镜3 — 叙事推进：** 自由景别，展现时间前后的事件

**提示词写作规范：**

1. **字数**：每段 80-120 字（中文字符）
2. **引用原图**：每段提示词开头必须包含"Based on the reference image,"或"保持参考图中人物的外貌特征，"等引用指令，明确告诉 AI 生成工具这是基于原图的变体，不是从零生成
3. **镜头起笔**：引用原图指令之后，紧接镜头动作/景别（从句式库中选取或自创，如"缓慢推进的中近景""极端俯拍的大远景"）
4. **内容按公式组织**：`[引用原图指令], [镜头类型&角度], [人物完整外观+动作], [有层次的场景描述], [光线设计], [色调/色彩分级], [情绪氛围]`
5. **人物描述**：每段必须完整写出人物外貌和服装，不可省略或写"同前""如上"
6. **一致性**：色调、光线方向、视觉风格、时间段必须与锚定镜头保持严格一致
7. **差异性**：3个分镜的景别、角度、构图必须有明显差异
8. **具体性**：用具体视觉元素代替模糊形容词（不写"美丽的"，写"被夕阳染成琥珀色的"）
9. **画面比例**：所有分镜必须与原图保持相同的宽高比

### 第 4 步：质量自检

输出前逐项检查，不通过则修改后再输出：

- [ ] 每段 80-120 字
- [ ] 以镜头动作/景别开头
- [ ] 人物描述完整且跨帧一致
- [ ] 视觉风格、色调、光线与原图一致
- [ ] 3个分镜景别有明显差异（远景 + 特写 + 自由景别）
- [ ] 无"同前""如上""与第一张相同"等省略写法
- [ ] 用具体视觉元素，无"美丽""漂亮""好看"等模糊形容词
- [ ] 每段可独立使用，直接复制粘贴到 AI 生成工具
- [ ] 画面比例与原图一致

---

## 输出格式

直接输出3段纯文字提示词，段间空一行。不加标签、编号、参数符号、markdown格式。每段提示词自身就包含对参考图的引用指令，用户拿到就能直接配合原图使用。

输出提示词之后，附上简短的参数建议：
- **画面比例**：与原图一致（如 16:9、3:4 等）
- **参考强度/相似度**：建议 0.6-0.75
