# 微信公众号排版规范 — 杂志风格

## 基础信息
- 模板风格: 白底蓝色杂志风
- 模板文件: `assets/templates/blue-minimal.html`

## 配色
| 用途 | 颜色值 |
|------|--------|
| 主题色 | `#0044FF` / `rgb(0, 68, 255)` |
| 正文色 | `#1A1A1A` |
| 标题色 | `#000000` |
| 背景色 | `#FFFFFF` |
| 辅助灰 | `#888888` |
| 水印蓝 | `rgba(0, 68, 255, 0.1)` |

## 字体层级
| 元素 | 字号 | 字重 | 行高 | 字间距 |
|------|------|------|------|--------|
| 大标题 | 32px | 900 | 1.2 | -1px |
| 章节标题 | 22px | 800 | 1.4 | -0.5px |
| 二级标题 | 16px | 800 | - | - |
| 正文 | 15px | 400 | 1.75 | 0.5px |
| 标签 | 14px | 600 | - | 2px |
| 说明文字 | 13px | 400 | - | 0.5px |
| 英文标签 | 11px | 600 | - | 2-3px |

## 字体族
`'Times New Roman', serif` 为主

## 正文规范
- 对齐: 两端对齐 (text-align: justify)
- 颜色: #1A1A1A
- 段间距: margin-bottom 16px
- 内边距: 左右 24px

## 可用组件
1. **头部 Banner** — 蓝底全宽，左年份右英文标签
2. **标签 + 大标题 + 副标题** — 蓝色标签 + 黑色大标题 + 灰色副标题
3. **蓝色分割线** — 4px 高，#0044FF，章节分隔
4. **正文段落** — 标准正文排版
5. **章节标题（水印编号）** — 56px 淡蓝水印数字 + 22px 标题
6. **二级标题** — 16px 蓝色加粗
7. **蓝色金句区块** — 蓝底白字，居中，主句 + 副句
8. **三栏信息卡片** — flex 三列，英文标签 + 中文关键词 + 说明
9. **目录卡片** — 灰底，两列编号列表
10. **结尾** — END 标记

## 排版原则
- 章节间用蓝色分割线 + 大留白分隔
- 金句用蓝色区块而非灰色引用框
- 水印数字起装饰+导航作用
- 强调用蓝色加粗或蓝色下划线
- 整体风格: 极简、杂志感、品牌色统一
