# 微信公众号排版指南

## 核心理念

Less is more，内容为王。公众号排版的目标是让读者舒适地阅读内容，而不是展示排版技巧。头部大号（槽边往事、三联周刊、GQ等）都采用极简风格。

## 排版参数速查表

| 元素 | 推荐值 |
|------|--------|
| 正文字号 | 14-15px（14px精致，15px轻松）|
| 注释字号 | 12-13px |
| 行间距 | 1.5-1.75倍 |
| 字间距 | 1-1.5px |
| 正文颜色 | #545454 或 #333333（避免纯黑#000000）|
| 注释颜色 | #888888 |
| 页边距 | 8或16（两端缩进）|
| 段落长度 | 3-4行/段 |
| 段落间隔 | 空一行 |

## 标题层级

### 不需要花哨的大标题样式

| 不要 | 要 |
|------|-----|
| 编辑器的花哨标题样式 | 加粗区分重点 |
| H1/H2/H3多层级标题 | 空行分隔段落 |
| 标题字号过大（20px+）| 小标题16px，正文14px |
| 多种标题装饰混用 | 统一简洁的风格 |

### 不同文章类型的处理

| 类型 | 标题建议 |
|------|----------|
| 情感/故事类 | 不用小标题，靠段落留白 |
| 干货/教程类 | 最多1级小标题 + 数字序号（01、02）|
| 深度分析类 | 0-1级小标题，保持阅读连贯 |
| 资讯播报类 | 不用小标题 |

### 替代大标题的方法

- **加粗**：重点内容加粗
- **空行**：段落之间空一行分隔
- **序号**：01、02、03（纯文字，不用样式）
- **字号微调**：小标题比正文大1-2px即可
- **颜色**：关键词换色（全文不超过3种颜色）

## 配色三原则

1. 全文不超过3种颜色
2. 正文统一一种颜色（推荐#333333或#545454）
3. 重点/品牌色只用于点缀

### 行业配色参考

| 行业 | 推荐色系 |
|------|----------|
| 科技 | 蓝色 |
| 餐饮 | 橙色/红色 |
| 商务 | 深蓝 |
| 美妆/时尚 | 粉色/亮色 |
| 文艺 | 低饱和度色 |

## 图文搭配

- 每3段文字配1张图
- 每300字插入重点加粗
- 每600字可放数据图表
- 图片尺寸推荐：900×500px
- 非必要情况下，一张头图即可

## 移动端优化要点

| 要点 | 说明 |
|------|------|
| 首行不缩进 | 手机屏幕窄，缩进显得不整齐 |
| 每行约21字 | 自然换行最佳 |
| 段落短 | 每段3-4行，不超过一屏 |
| 多预览 | 发布前用手机预览效果 |

## 技术限制（避坑指南）

### 不支持的CSS

| 属性 | 替代方案 |
|------|----------|
| position定位 | 用margin/padding |
| 百分比单位 | 用px或vw/vh |
| 外部CSS | 只能用内联样式 |
| ID选择器 | 用class或内联 |
| JavaScript | 不支持 |

### 常见坑

- 白色背景用 `#fefefe`（不要用#ffffff，会被过滤）
- `padding` 比 `margin` 更容易生效
- SVG内嵌图像必须用素材库链接

## 封面图尺寸

| 类型 | 尺寸 | 比例 |
|------|------|------|
| 头条封面 | 900×383px | 2.35:1 |
| 次条封面 | 200×200px | 1:1 |
| 分享缩略图 | 500×500px | 1:1 |

## 提升完读率的技巧

1. 开头3秒抓住注意力
2. 段落短，避免大段文字堆砌
3. 留白充足，增加呼吸感
4. 重点加粗，方便跳读
5. 文末设置互动（提问、投票）

## 2026年趋势

- AI全流程赋能：30秒智能排版
- 深度内容价值化：用户愿为优质内容付费
- 极简风格主导：花哨样式被淘汰
- 完读率权重提升：算法更看重阅读质量

## 黄金法则

- 呼吸感 > 结构感
- 内容 > 样式
- 简洁 > 花哨
