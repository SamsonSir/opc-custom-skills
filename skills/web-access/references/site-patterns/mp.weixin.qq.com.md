## 平台特征

- **架构**: 静态 HTML + 部分 JS，内容在页面源码中
- **反爬强度**: 中等，有基本的反爬机制
- **登录要求**: 一般不需要登录即可阅读
- **内容有效期**: 文章链接可能有过期时间参数

## 有效模式

### 直接访问文章链接
如果已有文章链接，直接使用 `web_fetch` 或 `browser` 访问：
```
https://mp.weixin.qq.com/s/[ARTICLE_ID]
```

### 从搜狗微信搜索（已弱化）
搜狗微信搜索已不再推荐，返回结果质量下降。

### 通过公众号主页
1. 访问公众号主页（需要获取主页链接）
2. 滚动加载历史文章列表
3. 点击文章进入详情

## 已知陷阱

| 问题 | 说明 |
|------|------|
| 链接过期 | 部分分享链接带时间戳参数，会过期 |
| 图片防盗链 | 文章内图片有 referer 检查 |
| 视频内容 | 视频需要浏览器环境才能播放 |

## 推荐策略

1. **有文章链接**: 先用 `web_fetch` 尝试，失败转 `browser`
2. **搜索内容**: 使用通用搜索引擎（Bing/Google）搜索"site:mp.weixin.qq.com 关键词"
3. **批量获取**: 需要登录公众号后台，使用 `browser` 操作
