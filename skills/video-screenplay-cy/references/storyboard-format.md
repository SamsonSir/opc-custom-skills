# 分镜脚本格式规范

## 标准 9 字段表格

每个镜头包含以下 9 个字段：

| 字段 | 说明 | 必填 |
|------|------|------|
| 镜号 | 顺序编号，从 1 开始 | ✅ |
| 景别 | 画面中主体的范围大小 | ✅ |
| 运镜 | 镜头的运动方式 | ✅ |
| 画面描述 | 画面中的视觉内容（需详细到可直接用于生成分镜图） | ✅ |
| 台词/旁白 | 该镜头中的对白或解说词 | ✅ |
| 音效/音乐 | 声音设计说明 | ⬜ |
| 时长(秒) | 镜头持续时间 | ✅ |
| 转场 | 与下一镜头的衔接方式 | ⬜ |
| 备注 | 额外的制作说明 | ⬜ |

## 景别详解

### 景别类型及适用场景

| 景别 | 英文缩写 | 画面范围 | 情感表达 | 典型用途 |
|------|---------|---------|---------|---------|
| 大远景 | ELS | 广阔环境，人物极小 | 壮阔、渺小、史诗感 | 开场建立世界观、片尾 |
| 远景 | LS | 完整环境，人物可辨 | 宏大感、距离感 | 建立场景、交代环境 |
| 全景 | WS/FS | 人物全身+部分环境 | 客观、交代 | 展示人物与环境关系 |
| 中全景 | MFS | 人物膝盖以上 | 适度客观 | 走动、肢体动作 |
| 中景 | MS | 人物腰部以上 | 叙事性最强 | 对话、互动、演示 |
| 中近景 | MCU | 人物胸部以上 | 较亲密 | 单人讲述、访谈 |
| 近景 | CU | 人物肩部以上 | 亲密、情感 | 表情特写、情感表达 |
| 特写 | ECU | 脸部/物体细节 | 冲击力、心理 | 关键道具、眼神、细节 |

### 景别选择指南

根据内容意图选择景别：

- **介绍一个人** → 全景（交代环境）→ 中景（正常讲述）
- **表达情感** → 近景 → 特写
- **展示操作/演示** → 中景 → 特写（操作细节）
- **多人对话** → 中景（双人）→ 中近景（单人反应）
- **场景转换** → 远景/全景（新场景建立）
- **开场** → 远景/全景 → 推至中景
- **结尾** → 中景 → 拉至远景/全景

## 运镜详解

### 运镜类型及适用场景

| 运镜 | 英文 | 镜头动作 | 表达效果 | 典型用途 |
|------|------|---------|---------|---------|
| 固定 | Static | 镜头不动 | 稳定、客观 | 对话、讲述、演示 |
| 推 | Push/Zoom In | 向主体靠近 | 聚焦、紧张、深入 | 强调重点、引起注意 |
| 拉 | Pull/Zoom Out | 远离主体 | 展开、释放、揭示 | 揭示全貌、场景结束 |
| 摇 | Pan | 水平/垂直转动 | 环顾、扫视、展示 | 展示环境、跟随动作 |
| 移 | Dolly/Track | 平行移动 | 陪伴、流动 | 行走、空间展示 |
| 跟 | Follow | 跟随主体 | 陪伴、紧随 | 跟随人物移动 |
| 升 | Crane Up | 向上升起 | 俯瞰、宏大 | 揭示场景、结尾升华 |
| 降 | Crane Down | 向下降落 | 接近、亲近 | 从全景到人物 |
| 甩 | Whip Pan | 急速转动 | 急切、爆发、转折 | 突然转折、紧急事件 |
| 环绕 | Orbit | 围绕主体旋转 | 审视、360度 | 产品展示、人物介绍 |

### 运镜选择指南

根据内容情绪和节奏选择运镜：

- **平稳讲述** → 固定
- **要强调某个细节** → 推
- **展示全貌/总结** → 拉
- **介绍环境/空间** → 摇 或 移
- **跟着人物走** → 跟
- **突然的转折** → 甩
- **产品/物品展示** → 环绕
- **大部分教程内容** → 固定为主，偶尔推/拉

## 转场效果

| 转场 | 英文 | 效果 | 适用场景 |
|------|------|------|---------|
| 切换 | Cut | 直接跳转 | 同场景内换角度、快节奏 |
| 淡入 | Fade In | 从黑/白渐现 | 开场、新章节开始 |
| 淡出 | Fade Out | 渐隐至黑/白 | 结尾、章节结束 |
| 叠化 | Dissolve | 前后画面叠加过渡 | 时间流逝、场景柔和切换 |
| 划变 | Wipe | 画面被推开 | 分步骤、并列对比 |
| 闪白 | Flash | 白色闪烁过渡 | 回忆、灵感闪现 |
| 匹配剪辑 | Match Cut | 相似构图/动作衔接 | 类比、前后对照 |

## 音效/音乐标注

常用标注方式：

- `BGM淡入` / `BGM淡出` / `BGM切换`：背景音乐变化
- `环境音：[描述]`：如"咖啡厅嘈杂声""键盘敲击声"
- `音效：[描述]`：如"叮咚提示音""快门声"
- `静音`：刻意的无声段落
- `音乐渐强` / `音乐渐弱`：音量变化

## 画面描述写法指南

画面描述是分镜表中最关键的字段——后续要直接根据它生成分镜图。每个画面描述应该是一段纯粹的自然语言，让人读完就能在脑中看到具体画面。

**必须包含的要素：**
- **人物**：引用角色表中的外观设定（穿着、发型、年龄），保持全片一致
- **环境**：具体的空间、物品、色调、光线
- **动作/姿态**：人物正在做什么、什么表情
- **构图**：人物在画面中的位置和占比

**好的画面描述：**
> 一位穿深灰色圆领卫衣的短发年轻男性坐在白色极简书桌前，面朝镜头自然微笑，右手轻放在银色笔记本电脑旁，桌面整洁只有电脑和一杯咖啡，背景是浅灰墙壁上一幅小尺寸绿植装饰画，柔和的暖色灯光从左上方打下，人物居中占画面三分之二

**不好的画面描述：**
> 主讲人坐在桌前，背景简洁

## 分镜密度参考

不同类型内容的每分钟镜头数参考：

| 内容类型 | 建议镜头密度 | 说明 |
|---------|------------|------|
| 教程/知识 | 1-2 镜/分钟 | 以内容清晰为主，少切换 |
| Vlog/旅行 | 3-5 镜/分钟 | 节奏活泼，场景多变 |
| 访谈/对话 | 2-3 镜/分钟 | 正反打切换 |
| 叙事/剧情 | 3-6 镜/分钟 | 根据情绪节奏调整 |
| 数据/展示 | 1-3 镜/分钟 | 数据图表需停留时间 |
